# ImSky

A Bluesky client made with Dear ImGui in C#.

This is under development, is full of bugs, and is not ready for daily use. You have been warned.

## Features

- [ ] Authentication
  - [x] Password authentication
  - [x] Session renewal
  - [ ] OAuth support
- [ ] Viewing posts
  - [x] "Following" feed
  - [x] Replies
  - [ ] Profile view
  - [x] Custom feeds
- [ ] Rich content
  - [ ] Embeds
    - [x] Images
    - [ ] GIFs
    - [ ] Videos
  - [ ] Mentions
  - [ ] Hashtags
  - [ ] Links
- [ ] Authoring posts
  - [x] New posts
  - [ ] Replies
  - [ ] Quote messages
  - [ ] Attaching embeds
  - [ ] Attaching facets
- [ ] Notifications
- [ ] Direct messages
